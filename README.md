## About the Project üìÉ

Warm Paws is a winter-themed pet care platform built to help pet owners provide better care during cold seasons.  
It offers essential services such as **Clothing**, **Comfort**, **Grooming**, **Health**, **Wellness**, and **Outdoor Care**, ensuring pets stay warm, safe, and healthy.

The platform is fully responsive and includes authentication, expert doctor information, customer reviews, smooth animations, and protected routes. From browsing services to managing profiles, Warm Paws delivers a clean and intuitive user experience.

## Installation ‚öôÔ∏è

Clone the repository and install dependencies:

```bash
git clone https://github.com/eistiakahmed/Warm-Paws.git
cd Warm-Paws
npm install

Set up environment variables by creating a `.env` file in the root directory:

```env
VITE_apiKey=your_key
VITE_authDomain=your_domain
VITE_projectId=your_project_id
VITE_storageBucket=your_bucket
VITE_messagingSenderId=your_id
VITE_appId=your_app_id

```
Run the application:

```bash
npm run dev
```
