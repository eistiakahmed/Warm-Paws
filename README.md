# Warm Paws - Pet Care in Winter

## Purpose

Warm Paws is a pet care service website designed for the winter season.It helps pet owners take better care of their furry friends during cold weather by providing various service categories such as Clothing, Comfort, Grooming, Health, Wellness, and Outdoor care.

## Key Features

- Winter-themed pet care service platform
- Fully responsive design for mobile, tablet, and desktop
- Firebase Authentication (Email/Password & Google Sign-In)
- Protected route for service details
- Password validation and Forgot Password functionality
- Profile update feature (Name & Image)
- Smooth animations using AOS and Animate.css
- Toast notifications for success/error feedback
- JSON-based service data fetching with Axios
- Clean and modern UI built with TailwindCSS and DaisyUI
